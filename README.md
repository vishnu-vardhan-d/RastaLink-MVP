# RastaLink

**Proof of concept · early-stage idea.**

A **UI prototype** of a load-matching and fleet-tracking concept for trucking across India (Kashmir to Kanyakumari), with AI-assisted matching planned. Its purpose is to **explore the experience and validate requirements — what users would actually want to see** — *before* building the real platform. Everything on screen runs on **mock / sample data**.

> **Disclaimer:** This is a **proof-of-concept UI prototype only** — there is **no working functionality** behind it (no backend, no database, no live data); every figure shown is **mock/sample data** used to explore requirements. It is **voluntary, unpaid work** on personal time, with **no commercial aspect and no employment** involved — not a product, not for sale. A fuller, functional version is being explored privately.

## What you're looking at

A preview interface only — live-style order feeds, load matching, fleet status, and a rate ticker — rendered from sample data so the experience is visible end-to-end. No backend, database, or login required to run it.

## Tech

React · Vite · TypeScript · Tailwind CSS

## Run locally

```bash
npm install
npm run dev      # http://localhost:5000
```
